# WatchDogTinyS3

## Usage
```
python WatchDogTinyS3.py /home/user/images
```
##Dependencies
- python 2.6 or above.
- watchdog 0.8.3
- tinys3 0.1.11
- logging *

##Installation

Installing from PyPI using pip:
```
$ pip install watchdog
$ pip install tinys3
$ pip install loggin
```
Installing from PyPI using easy_install:
```
$ easy_install watchdog
$ easy_install tinys3
$ easy_install loggin
```
